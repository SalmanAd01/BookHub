# BookHub
Link : https://bookkhub.herokuapp.com/
React: https://book-1g9aq7tlq-salmanad01.vercel.app/
#### Get Started
copy the .env.exmaple to .env

```
  cp .\.env.example .\.env
```

To Install dependencies

```
    go mod download
```

To run 

```
    air
```

#### Rus using docker

copy the .env.exmaple to .env

```
  cp .\.env.example .\.env
```

Build and run the image
```
   docker compose -f .\docker-compose-dev.yml up
```

### To Run Lint
```
  golangci-lint run
```
